My HTML notes

What is Markup?

Is what we use to format page elements.

Everything, we put on a page, that we want to show with a specific format rather than simple text, we place inside html tags.

Hyperlinks

These are added using the anchor tag <a> . You will also need to add a hypertext reference known as a href attribute inside the opening a tag to give it a destination, the url. Like this

<a href="http://somenicewebsite.com">The Website</a>

Like this you can add a link somewhere, like this: Google

Images

For images you do something similar using a image tag <img>. Instead of a URL we add a file path. The image tag takes a src attribute so we know where to get the image from.

It is also god practice to add alt text to an image so screenreaders can tell what it is. The code will look like this:

<img="images/wolves.jpg" alt="some nice wolves">
